1. Introducción a Java

Java es un lenguaje de programación orientado a objetos, ampliamente utilizado en desarrollo de aplicaciones web y de escritorio.

1.1 Tu Primer Programa

public class Main {
  public static void main(String[] args) {
    System.out.println("¡Hola, mundo!");
  }
}
Salida: ¡Hola, mundo!

2. Variables y Tipos de Datos

2.1 Variables

En Java, las variables almacenan información de distintos tipos.

String nombre = "Ana";
int edad = 25;
double altura = 1.65;
boolean esEstudiante = true;

System.out.println("Me llamo " + nombre + " y tengo " + edad + " años");

3. Estructuras de Control

3.1 Condicionales if-else

Las estructuras condicionales permiten ejecutar diferentes bloques de código dependiendo de si una condición se cumple o no. El ejemplo más común es el condicional if-else.

edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

3.2 Bucles

Bucle for

El bucle for se usa para repetir una acción un número determinado de veces. Es muy útil cuando se sabe cuántas iteraciones se deben hacer.

for i in range(5):
    print(f"Número: {i}")

Bucle while

El bucle while se ejecuta mientras se cumpla una condición. Es útil cuando no se sabe cuántas iteraciones se necesitarán de antemano.

contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1

4. Funciones

4.1 Definición de Funciones

Las funciones son bloques de código reutilizables que se pueden llamar para realizar una tarea específica. Esto mejora la modularidad del código y evita la repetición.

def saludar(nombre):
    return f"¡Hola, {nombre}!"

mensaje = saludar("Ana")
print(mensaje)

4.2 Parámetros y Retorno

Las funciones pueden recibir parámetros (valores de entrada) y devolver resultados. Esto permite crear funciones más dinámicas y útiles.

def suma(a, b):
    return a + b

resultado = suma(5, 3)
print(f"La suma es: {resultado}")

5. Listas y Tuplas

5.1 Listas

Las listas son estructuras que permiten almacenar múltiples elementos. Son dinámicas, lo que significa que se pueden modificar, agregar o eliminar elementos en cualquier momento.

frutas = ["manzana", "banana", "naranja"]
frutas.append("pera")
print(frutas[0]) # Primera fruta
print(len(frutas)) # Cantidad de frutas

5.2 Tuplas

Las tuplas son similares a las listas, pero a diferencia de ellas, las tuplas son inmutables. Esto significa que no pueden modificarse una vez creadas.

coordenadas = (10, 20)
x, y = coordenadas
print(f"X: {x}, Y: {y}")

6. Diccionarios

Los diccionarios permiten almacenar pares clave-valor. Son muy útiles cuando necesitas asociar un dato con otro, como un nombre con un número de teléfono.

persona = {"nombre": "Juan", "edad": 30, "ciudad": "Madrid"}
print(persona["nombre"]) # Imprime 'Juan'

7. Manejo de Excepciones

El manejo de excepciones se utiliza para capturar errores en el código y gestionarlos adecuadamente sin que el programa se detenga abruptamente.

try:
    numero = int(input("Introduce un número: "))
except ValueError:
    print("¡Error! No es un número válido.")

8. Clases y Objetos

La programación orientada a objetos (OOP) permite crear clases que sirven como plantillas para crear objetos. Un objeto es una instancia de una clase.

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
persona1 = Persona("Ana", 25)
print(persona1.nombre) # Imprime 'Ana'

9. Manejo de Archivos

Python permite trabajar con archivos de texto para leer y escribir datos de manera persistente. Es muy útil para guardar configuraciones o información generada por el programa.

with open("archivo.txt", "w") as f:
    f.write("Hola Mundo")
with open("archivo.txt", "r") as f:
    contenido = f.read()
    print(contenido)

10. Uso de Librerías Externas

Las librerías externas proporcionan funcionalidades adicionales. Puedes instalarlas con pip y usarlas para añadir características como la manipulación de fechas, conexión a bases de datos, entre otros.

import requests
response = requests.get("https://api.example.com")
print(response.json())
