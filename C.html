1. Introducción a C+++

C+++ es un lenguaje de programación basado en C que incorpora características avanzadas como clases, manejo de excepciones y más. Es utilizado tanto para aplicaciones de sistemas como de software de alto rendimiento.

1.1 Tu Primer Programa en C+++

#include <iostream>
using namespace std;

int main() {
    cout << "¡Hola, mundo!" << endl;
    return 0;
}
Salida: ¡Hola, mundo!

1.2 El Compilador de C+++

C+++ es un lenguaje compilado, por lo que necesitas un compilador como GCC o Clang para ejecutar tus programas.

2. Variables y Tipos de Datos

2.1 Variables

Las variables en C+++ se usan para almacenar datos de diferentes tipos.

string nombre = "Ana";
int edad = 25;
double altura = 1.65;
bool es_estudiante = true;
cout << "Me llamo " << nombre << " y tengo " << edad << " años." << endl;

2.2 Tipos de Datos Básicos

  • int (números enteros)
  • float (números decimales)
  • double (números de doble precisión)
  • char (caracteres)
  • bool (valores booleanos)
  • string (cadenas de texto)

3. Estructuras de Control

3.1 Condicionales if-else

int edad = 18;
if (edad >= 18) {
    cout << "Eres mayor de edad" << endl;
} else {
    cout << "Eres menor de edad" << endl;
}

3.2 Bucles

Los bucles permiten ejecutar un bloque de código varias veces.

for (int i = 0; i < 5; i++) {
    cout << i << " ";
}

4. Funciones

Las funciones en C+++ son bloques de código reutilizables. Definimos una función y la llamamos cuando la necesitemos.

int suma(int a, int b) {
    return a + b;
}

int main() {
    cout << suma(3, 4) << endl;
    return 0;
}

5. Arreglos y Vectores

Los arreglos (o vectores) son estructuras que permiten almacenar múltiples valores del mismo tipo. En C+++, los arreglos tienen un tamaño fijo, y su acceso se hace mediante índices.

5.1 Arreglos Unidimensionales

int edades[5] = {18, 25, 30, 22, 35};
for (int i = 0; i < 5; i++) {
    cout << edades[i] << " ";
}

5.2 Arreglos Multidimensionales

int matriz[2][2] = {{1, 2}, {3, 4}};
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 2; j++) {
        cout << matriz[i][j] << " ";
    }
}

6. Punteros

Los punteros son variables que almacenan la dirección de memoria de otras variables. Son muy útiles para manipular datos en memoria directamente.

6.1 Declaración y Uso de Punteros

int a = 5;
int* ptr = &a;
cout << "Valor de a: " << a << endl;
cout << "Dirección de a: " << &a << endl;
cout << "Valor de ptr: " << ptr << endl;
cout << "Valor apuntado por ptr: " << *ptr << endl;

6.2 Punteros y Arreglos

int arreglo[3] = {10, 20, 30};
int* ptr = arreglo;
for (int i = 0; i < 3; i++) {
    cout << *(ptr + i) << " ";
}

7. Programación Orientada a Objetos (OOP)

La programación orientada a objetos (OOP) permite organizar el código mediante clases y objetos. OOP facilita la reutilización y el mantenimiento del código.

7.1 Clases y Objetos

class Persona {
public:
    string nombre;
    int edad;
    Persona(string n, int e) { nombre = n; edad = e; }
    void mostrar() { cout << nombre << " tiene " << edad << " años." << endl; }
};

int main() {
    Persona p("Juan", 30);
    p.mostrar();
}

7.2 Herencia

class Estudiante : public Persona {
public:
    string carrera;
    Estudiante(string n, int e, string c) : Persona(n, e) { carrera = c; }
    void mostrar() {
        cout << nombre << " estudia " << carrera << endl;
    }
};

int main() {
    Estudiante e("Ana", 22, "Ingeniería");
    e.mostrar();
}

8. Manejo de Excepciones

Las excepciones permiten manejar errores en tiempo de ejecución de manera más eficiente. En C+++, se usan bloques try-catch.

8.1 Manejo Básico de Excepciones

try {
    int a = 10;
    int b = 0;
    if (b == 0) throw "División por cero";
    cout << a / b << endl;
} catch (const char* msg) {
    cout << "Error: " << msg << endl;
}

9. Plantillas (Templates)

Las plantillas en C+++ permiten crear funciones y clases que operan con diferentes tipos de datos.

9.1 Funciones Template

template <typename T>
T suma(T a, T b) {
    return a + b;
}

int main() {
    cout << suma(5, 3) << endl;
    cout << suma(2.5, 3.5) << endl;
}

10. Archivos

El manejo de archivos en C+++ permite leer y escribir datos desde o hacia archivos en disco. Usamos las librerías fstream para leer y escribir archivos.

10.1 Lectura de Archivos

#include <fstream>
#include <iostream>

int main() {
    fstream archivo("datos.txt", ios::in);
    if (archivo.is_open()) {
        string linea;
        while (getline(archivo, linea)) {
            cout << linea << endl;
        }
    archivo.close();
}
}

10.2 Escritura en Archivos

#include <fstream>
#include <iostream>

int main() {
    fstream archivo("output.txt", ios::out);
    if (archivo.is_open()) {
        archivo << "Hola, mundo!" << endl;
        archivo.close();
}
}
