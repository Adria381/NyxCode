1. Introducción a C#

C# es un lenguaje de programación desarrollado por Microsoft. Es muy popular para aplicaciones de escritorio, videojuegos (con Unity) y servicios web.

1.1 Tu Primer Programa

using System;

class Program
{
  static void Main()
  {
    Console.WriteLine("¡Hola, mundo!");
  }
}
Salida: ¡Hola, mundo!

2. Variables y Tipos de Datos

2.1 Variables

Las variables en C# se utilizan para almacenar datos.

string nombre = "Ana";
int edad = 25;
double altura = 1.65;
bool esEstudiante = true;

Console.WriteLine($"Me llamo {nombre} y tengo {edad} años");

2.2 Tipos de Datos Básicos

  • string (cadenas de texto)
  • int (números enteros)
  • double (números decimales)
  • bool (valores booleanos)

3. Estructuras de Control

3.1 Condicionales if-else

int edad = 18;
if (edad >= 18)
    Console.WriteLine("Eres mayor de edad");
else
    Console.WriteLine("Eres menor de edad");

3.2 Bucles

Bucle for

for (int i = 0; i < 5; i++)
    Console.WriteLine(i);

Bucle while

int i = 0;
while (i < 5)
    {
      Console.WriteLine(i);
      i++;
    }

6. Clases y Objetos

6.1 Definición de una Clase

class Persona
{
  public string Nombre;
  public int Edad;
}

Persona persona = new Persona();
persona.Nombre = "Ana";
persona.Edad = 25;
Console.WriteLine(persona.Nombre); // Ana

7. Herencia

7.1 Clases Derivadas

class Animal
{
  public void Hablar()
  {
    Console.WriteLine("Sonido de Animal");
  }
}

class Perro : Animal
{
  public void Hablar()
  {
    Console.WriteLine("Guau!");
  }
}

Perro perro = new Perro();
perro.Hablar(); // Guau!

8. Interfaces

8.1 Definición y Uso de Interfaces

interface IAnimal
{
  void Hablar();
}

class Gato : IAnimal
{
  public void Hablar()
  {
    Console.WriteLine("Miau!");
  }
}

IAnimal gato = new Gato();
gato.Hablar(); // Miau!

11. LINQ

11.1 Consultas LINQ

int[] numeros = {1, 2, 3, 4, 5};
var resultado = from n in numeros
  where n > 2
  select n;

foreach (var num in resultado)
  Console.WriteLine(num);
