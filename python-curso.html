1. Introducción a Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Es conocido por su sintaxis clara y legible, lo que lo hace perfecto para principiantes.

1.1 Tu Primer Programa

print("¡Hola, mundo!")
Salida: ¡Hola, mundo!

1.2 El Intérprete de Python

Python es un lenguaje interpretado, lo que significa que puedes escribir y ejecutar código línea por línea.

Tip

Puedes usar IDLE, PyCharm o VSCode para escribir y ejecutar código Python.

2. Variables y Tipos de Datos

2.1 Variables

Las variables son contenedores para almacenar datos.

nombre = "Ana"
edad = 25
altura = 1.65
es_estudiante = True

print(f"Me llamo {nombre} y tengo {edad} años")

2.2 Tipos de Datos Básicos

  • str (cadenas de texto)
  • int (números enteros)
  • float (números decimales)
  • bool (valores booleanos)

Ejercicio Práctico

Crea variables para almacenar tu información personal y muéstralas usando print().

3. Estructuras de Control

3.1 Condicionales if-else

edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

3.2 Bucles

Bucle for

for i in range(5):
    print(f"Número: {i}")

Bucle while

contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1

4. Funciones

4.1 Definición de Funciones

def saludar(nombre):
    return f"¡Hola, {nombre}!"

mensaje = saludar("Ana")
print(mensaje)

4.2 Parámetros y Retorno

def suma(a, b):
    return a + b

resultado = suma(5, 3)
print(f"La suma es: {resultado}")

5. Listas y Tuplas

5.1 Listas

frutas = ["manzana", "banana", "naranja"]
frutas.append("pera")
print(frutas[0]) # Primera fruta
print(len(frutas)) # Cantidad de frutas

5.2 Tuplas

coordenadas = (10, 20)
x, y = coordenadas
print(f"X: {x}, Y: {y}")
