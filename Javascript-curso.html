1. Introducción a JavaScript

JavaScript es un lenguaje de programación esencial en el desarrollo web. Permite crear interactividad y manejar datos en tiempo real en aplicaciones web.

1.1 Tu Primer Programa

console.log("¡Hola, mundo!");
Salida: ¡Hola, mundo!

2. Variables y Tipos de Datos

2.1 Variables

Las variables en JavaScript se pueden declarar con var, let, o const.

let nombre = "Ana";
const edad = 25;
let altura = 1.65;
let esEstudiante = true;

console.log(`Me llamo ${nombre} y tengo ${edad} años`);

3. Estructuras de Control

3.1 Condicionales if-else

let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

4. Funciones

4.1 Definición de Funciones

function saludar(nombre) {
  return `¡Hola, ${nombre}!`;
}

const mensaje = saludar("Ana");
console.log(mensaje);

5. Arreglos y Objetos

5.1 Arreglos

let frutas = ["manzana", "banana", "naranja"];
frutas.push("pera");
console.log(frutas[0]); // Primera fruta
console.log(frutas.length); // Cantidad de frutas
