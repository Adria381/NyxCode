1. Introducción a HTML

HTML es el lenguaje base para crear contenido en la web. Se usa para estructurar y organizar el contenido en las páginas.

1.1 Ejemplo Básico

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>

¡Hola Mundo!

2.1 Encabezados, Párrafos y Enlaces

3. Introducción a CSS

CSS es el lenguaje usado para dar estilo y diseño a tu contenido HTML.

3.1 Cambiando Colores y Fuentes

<style>
  body {
    background-color: #f0f4f8;
    font-family: Arial, sans-serif;
  }
</style>

4. Selectores CSS

Los selectores en CSS permiten aplicar estilos a elementos específicos en la página.

4.1 Selectores de Clase y ID

<style>
  .miClase { color: blue; }
  #miID { font-weight: bold; }
</style>

<p class="miClase">Texto azul</p>
<p id="miID">Texto en negrita</p>

5. Layout y Posicionamiento

Aprenderás a crear diseños de página usando CSS Flexbox y Grid para organizar los elementos de manera flexible y eficiente.

5.1 Ejemplo de Flexbox

Con Flexbox, los elementos se distribuyen automáticamente dentro de un contenedor flexible. Esto es útil para crear alineaciones horizontales o verticales.

<style>
  .flex-container {
    display: flex;
    gap: 10px;
  }
  .flex-item {
    padding: 20px;
    background-color: #4A90E2;
    color: #fff;
    text-align: center;
  }
</style>

<div class="flex-container">
  <div class="flex-item">Elemento 1</div>
  <div class="flex-item">Elemento 2</div>
  <div class="flex-item">Elemento 3</div>
</div>

Elemento 2
Elemento 3

5.2 Ejemplo de CSS Grid

CSS Grid permite organizar los elementos en filas y columnas, ideal para crear diseños de cuadrícula complejos.

<style>
  .grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
  }
  .grid-item {
    padding: 20px;
    background-color: #D64541;
    color: #fff;
    text-align: center;
  }
</style>

<div class="grid-container">
  <div class="grid-item">Elemento A</div>
  <div class="grid-item">Elemento B</div>
  <div class="grid-item">Elemento C</div>
  <div class="grid-item">Elemento D</div>
  <div class="grid-item">Elemento E</div>
  <div class="grid-item">Elemento F</div>
</div>
